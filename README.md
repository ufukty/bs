# bs

Basic server. No bs.

## Install

```sh
go install github.com/ufukty/bs
```

## Usage

Supports nothing but port and directory flags.

```sh
bs [ -p <port> ] [ -d <directory> ]
```

they default to `8080` and `.` so most of the time:

```sh
bs
```

Then go to [127.0.0.1:8080] in your browser.
